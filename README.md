AI House Price Prediction
PROJECT OVERVIEW:
  An AI-powered House Price Prediction System built using Python, Scikit-Learn, and Streamlit. The application predicts house prices based on property features and provides     an interactive dashboard with property summaries, AI recommendations, and suggested house blueprints.
FEATURES:
  Predict house prices using Machine Learning
  Interactive Streamlit dashboard
  AI-based property recommendation
  Suggested house blueprint
  Property summary dashboard
  Clean and user-friendly interface
TECHNOLOGIES USED:
  Python
  Streamlit
  Scikit-Learn
  Pandas
  NumPy
  Joblib
  Pillow
